# プライバシーポリシー

**二級ボイラー技士Pass**（以下、「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の適切な取り扱いに努めます。本プライバシーポリシーは、本アプリにおける情報の収集・利用・管理方針について説明するものです。

最終更新日：2026年6月19日

## 1. 収集する情報

### 端末内に保存する情報

本アプリは、以下の学習データを端末内に保存します。

- 問題への回答履歴、正誤記録
- 苦手フラグ（ブックマーク）
- 連続学習日数
- 文字サイズなどの設定
- 広告視聴による追加解放情報

これらの学習データは、原則として端末内に保存され、当社が運営する独自アカウントサーバーには送信されません。

### Firebase Analyticsで送信する情報

本アプリは、利用状況の分析と品質改善のため、Firebase Analyticsを利用します。Firebase Analyticsでは、以下のような情報が送信される場合があります。

- アプリの起動、学習開始、回答、学習完了、模試開始・完了などのイベント
- プレミアム案内の表示、購入開始、購入完了、レビュー依頼候補などのイベント
- アプリのバージョン、端末種別、OSバージョン、おおよその地域などの技術情報
- アプリの操作状況、パフォーマンス、診断に関する情報

Firebase Analyticsには、問題本文、選択肢本文、解説本文、自由入力、記述答案、氏名、メールアドレス、個別相談内容、question_id、RevenueCat app_user_id、Apple transaction IDの生値、RevenueCat SDK Key、AdMob ID、公式問題本文や出題文を復元できる情報を送信しません。

本アプリのFirebase Analyticsは、iOSで広告ID（IDFA）を利用しない構成（withoutAdIdSupport）で導入しています。ATT（App Tracking Transparency）の許可要求は行いません。

### Google AdMob（広告配信）

無料ユーザー向けに広告を表示するため、Google AdMobを利用しています。AdMobでは、広告配信、広告効果測定、不正防止などのため、端末情報、広告識別子、IPアドレス、アプリの使用状況などが利用される場合があります。

本アプリでは、非パーソナライズ広告を前提とした配信設定を行っています。

Googleのプライバシーポリシー：https://policies.google.com/privacy

### RevenueCat（課金管理）

アプリ内課金の購入状態の確認、復元、管理のため、RevenueCatを利用しています。RevenueCatでは、購入状態の管理に必要な匿名識別子や購入履歴が処理される場合があります。

RevenueCatのプライバシーポリシー：https://www.revenuecat.com/privacy

## 2. 利用目的

収集または処理される情報は、以下の目的で利用します。

- 問題演習、復習、学習進捗表示の提供
- 誤答優先学習、苦手問題、連続学習日数などの学習機能の提供
- アプリの利用状況分析と品質改善
- 広告表示およびリワード広告による問題追加解放
- アプリ内課金の購入状態の確認、復元、管理
- 不具合調査、パフォーマンス改善、ストア要件への対応

## 3. 利用していないFirebaseサービス

本アプリでは、以下のFirebaseサービスは利用していません。

- Firebase Auth
- Firestore
- Realtime Database
- Cloud Functions
- Storage
- BigQuery export
- Firebase AI Logic
- Remote Config
- Crashlytics
- App Check
- Cloud Messaging

## 4. 取得しない情報

本アプリは、利用者の氏名、住所、電話番号、メールアドレス、連絡先、正確な位置情報、健康情報、設備診断や個別相談の内容を、アカウント情報として独自に取得しません。

## 5. 情報の第三者提供

本アプリは、法令に基づく場合を除き、ユーザーの情報を第三者に販売しません。Firebase Analytics、Google AdMob、RevenueCat、Appleが提供するApp Storeおよびアプリ内課金の仕組みにおける情報の取り扱いは、それぞれの事業者のポリシーに従います。

## 6. 情報の保管と削除

学習データは端末内に保存されます。アプリ内の「設定」から学習データをリセットすると、回答履歴、苦手フラグ、連続学習日数、広告視聴による解放情報を削除できます。レビュー依頼の表示制御履歴は、同一バージョン内で過度にレビュー依頼を表示しないため、学習データリセットでは削除しません。

アプリをアンインストールすると、端末内に保存されたローカルデータは削除されます。
